Image Processing API

A fast and scalable API to process (resize/format/convert) images

TO USE IT : http://localhost:3000/resize/?name=fjord.jpg&h=400

GET: "/resize?name={NAME.jpg}&w={WIDTH}&h={HEIGHT}&ext=jpg"
GET: "/resize?name={NAME.jpg}&w={WIDTH}&h={HEIGHT}"

Introduction

This is a node JS application built with typescript consisting of an image processing API that takes an image file name and its dimension (width and height) as input. On the server it checks if the image with the requested dimensions is available or not. if the image is available then it returns it otherwise it converts the image to the requested size using the sharp.js library and then cache it on the server so that it can be sent back to the client when next time the same image is requested without having to process it again.

Basic API information:

API Base URL http://localhost:3000/resize This is a read-only API

Endpoint: /resize

Method: GET

Query Parameters:

name: string

w: number refer to width

h: number refer to height

ext: string [Optional] refer to fileExtension

How to Use?

Change the resizing behavior
By default, images are resized to fit into the specified dimensions if you input one dimension for example (w) only or (h)only ex: http://localhost:3000/resize/?name=fjord.jpg&h=400.
You can request a different behavior using the width or height parameter.
